# vim-configuration

## Install
    git clone git@github.com:futureinsky/.vim.git ~
* VIM:   ln -s ~/.vim/.vimrc ~/.vimrc
* NeoVim: ln -s ~/.vim/.vimrc ~/.nvim/init.vim


## Plugins
 Details see: .vim/vim-plug

    -> Nerdtree
    -> Tagbar
    -> Git
    -> FZF
    -> YouCompleteMe
    -> Markdown
